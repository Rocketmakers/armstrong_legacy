# Form collections
Armstrong also provides some standard ways of laying out forms, because lets face it, nobody enjoys laying out forms.

## Usage example
#### MARKUP
``` html
<div class="form">
  <div class="form-row">
    <label>Username</label>
    <input type="text" />
  </div>
  <div class="form-row">
    <label>Password</label>
    <input type="password" />
  </div>
  <div class="form-row">
    <input type="submit" value="Submit" />
  </div>
</div>
```
#### GENERATES
