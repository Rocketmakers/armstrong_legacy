# Progress indicators
A spinner and a progress bar. Progress bar is indeterminate by default, but just set its width to change its value

## Usage example
#### MARKUP
```
<div class="spinner"></div>
<br/>
<div class="progress-bar">
    <div style="width: 75%"></div>
</div>
```
#### GENERATES
<div class="example-code">
<div class="spinner"></div>
<br/>
<div class="progress-bar">
    <div style="width: 75%"></div>
</div>
</div>
