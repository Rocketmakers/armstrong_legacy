# Tables
Not the most revolutionary or exciting control, but they are still a required component for some sites, so lets tidy them up a bit so you don't have to worry!

## Additional classes

### table

#### .striped
Some basic zebra striping for your tables.

#### .borders
Puts visible borders around the cells

## Usage example
#### MARKUP
``` html
<table class="striped">
       <thead>
           <tr>
               <th>Heading 1</th>
               <th>Heading 2</th>
               <th>Heading 3</th>
               <th>Heading 4</th>
           </tr>
       </thead>
       <tbody>
           <tr>
               <td>Item 1</td>
               <td>Item 2</td>
               <td>Item 3</td>
               <td>Item 4</td>
           </tr>
           <tr>
               <td>Item 1</td>
               <td>Item 2</td>
               <td>Item 3</td>
               <td>Item 4</td>
           </tr>
           <tr>
               <td>Item 1</td>
               <td>Item 2</td>
               <td>Item 3</td>
               <td>Item 4</td>
           </tr>
       </tbody>
   </table>

```
